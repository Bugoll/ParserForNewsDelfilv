#Руководство по установке

1. Скопируйте себе проект.
2. Выполните команду создания окружения:
    
    Для Windows:
    $python -m venv venv
    $venv\Scripts\activate

    Для OS:
    $python3 -m venv venv
    $source venv/bin/activate


3. Установите завимимости проекта:

    (venv) $pip install -r requirements.txt
